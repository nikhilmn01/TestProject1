The project e2e contains the following folders and files

1. base_e2e (contains the base class where the webDriver is defined)
2. features_e2e (contains the feature files (appliTools.feature is the main file))
3. hooks_e2e (contains the hooks file for the project (browserInit_e2e.cs is the main file))
4. PageObjects_e2e (contains the page objects for the step definition (appliTools_pageObjects is the main file))
5. StepDefinition_e2e (contains stepDefinition for the project (AppliToolsStepDefinition.cs is the main file))
6. temp (folder to be ignored)

1. index.html contains the report of execution
